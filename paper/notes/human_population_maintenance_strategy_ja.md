# Human-Side Population Maintenance Strategy

## 位置づけ

このノートは、visibility radius simulation の結果を、人類側の行動・制度・UI設計に翻訳するための作業メモである。

ここでの主張は人口予測ではない。Simが示したのは、次の抽象メカニズムである。

```text
候補プール拡大
+ 非行動可能な比較対象
+ 限られた注意枠
+ 相対上位志向
-> 現実に関係形成できる候補が注意枠から落ちる
-> 未マッチ率上昇・出生 proxy 低下
```

この結果を現実にそのまま当てはめるのではなく、「文明退化なしに人口維持を狙うなら、どの介入がモデル上の失敗経路を塞ぐか」として読む。

## 結論

文明を退化させる必要はない。SNS、検索、推薦、グローバル可視性を捨てる必要もない。

必要なのは、仮想世界の比較能力を、現実世界の行動可能性へ接続し直すことである。

Simの用語で言えば、最も直接的な対策は `actionable_selection_reserve` の社会実装である。つまり、無限に見える上位候補ではなく、実際に会える、生活圏が重なる、相互関心がある、時間制約内で関係が進む候補を、選択枠の中に必ず残す。

## 人類側の実装案

### 1. Actionable reserve をUIに入れる

マッチング、SNS、紹介、イベント設計で、候補提示を単なる魅力度ランキングにしない。

優先する軸は次の通り。

- 実際に会える距離
- 生活圏の重なり
- 相互関心
- 返信・会話・再会の成立確率
- 年齢と時間制約に対する現実性
- 長期協力の見込み

これは「ローカルに戻れ」という意味ではない。グローバルな情報空間は維持しつつ、家族形成に関わる選択では、非行動可能な比較対象が全ての注意枠を奪わないようにする。

### 2. 仮想上位比較を候補ではなくノイズとして扱う文化を作る

SNS上で見える上位個体は、しばしば実際の候補ではない。

広告、演出、アルゴリズム選抜、遠距離、既婚・非関心・非現実的接点などが混じる。これを「候補」と誤認すると、現実に成立する候補が不当に低く見える。

必要なのは道徳説教ではなく、仮想世界リテラシーである。

```text
見えている上位個体
!= 自分の候補集合
```

この認知を個人任せにせず、教育、メディア、UI、地域コミュニティ、友人紹介の慣習として支える。

### 3. 20〜30代の時間制約を前提に設計する

学習やフィードバックは遅い。適齢期後半に「上位比較は現実候補ではなかった」と気づいても、人口維持には間に合いにくい。

したがって、探索コストを下げ、反復接触を増やす仕組みが必要になる。

- 大学・職場・地域での自然な反復接触
- 信頼できる紹介
- 目的が露骨すぎない共同活動
- 初回接触より、2回目以降が発生しやすい設計
- 失敗しても生活圏が壊れない低摩擦な場

ポイントは、短期的なマッチ数ではなく、関係が育つ確率を上げることである。

### 4. 子どもを持つコストを下げる

これはペア形成後の出生を支える領域である。

OECDやUNFPAの整理でも、低出生率への対応として、育児休業、保育、住宅、安定雇用、父親の育児参加、仕事と家庭の両立が重視されている。単発の現金給付だけで十分とは見なされにくい。

ただし、本Simの観点では、これは「成立したペアが子どもを持てるようにする政策」であり、候補プール拡大によるペア形成崩壊そのものには、別の対策が要る。

### 5. ランキング市場から関係市場へ寄せる

恋愛・結婚を上位候補のランキング市場にすると、勝者総取りに近づく。

人口維持に必要なのは、全員が理想上位を追うことではなく、十分に多くの人が現実に成立する関係を作れることだ。

したがって評価軸は、見た目の上位性や瞬間的魅力度だけでなく、相性、信頼、生活互換性、協力性、継続性を強く含む必要がある。

## 移民と制度差の扱い

移民は人口構造を支える強い要素だが、先進文明全体が同じ仮想比較環境に入るなら、根本解というより時間を買う手段になる可能性がある。

制度差は重要だが、今のSimでは明示していない。US/UKの人口増や地域差は、`birth_probability`、`carrying_capacity`、`lifespan_mean`、`pair_duration_mean` などのキャリブレーションパラメータに吸収されている。

次に制度差を入れるなら、大きな制度モデルを作るより先に、次の2つを薄く足すのがよい。

- 移民: 年齢層つきの外部流入率
- 制度: ペア形成成功率または出生確率への地域係数

この程度なら、仮説の本体である「候補プール拡大 x 注意枠 x 上位志向」を壊さずに比較できる。

## 研究上の次アクション

次の検証では、人口維持策をSim上の介入として比較する。

- `actionable_selection_reserve_fraction` をUI・文化介入の proxy として国別に掃く
- 反復接触を増やす介入として、local candidate revisit や pair formation memory を入れる
- 外部流入率を足し、移民が「時間を買う」効果と「根本解ではない」可能性を分ける
- 制度係数を出生確率とペア形成確率に分け、どちらが効くかを見る
- 20〜30代の時間制約をより強くし、学習遅延の影響を測る
